# ClearLedger AI 💸🤖

An AI-powered personal finance management and fraud detection platform built for students and young professionals.

ClearLedger AI helps users:
- Track expenses and savings
- Monitor subscriptions
- Analyze financial trends
- Detect suspicious transactions using AI
- Visualize spending through dashboards

---

# 🚀 Features

## 👤 Authentication
- Login Page
- Register Page
- Secure User Access

## 📊 Dashboard
- Financial overview
- Expense tracking
- Savings insights
- Analytics charts

## 🤖 AI Assistant
- AI-powered finance chatbot
- Smart financial suggestions
- Budget recommendations

## 🛡 Fraud Detection
- Detect suspicious transactions
- AI-based risk prediction
- Fraud alert system

## 📈 Analytics
- Spending visualization
- Reports and statistics
- Monthly financial analysis

## 💳 Subscription Tracker
- Track recurring AI tool subscriptions
- Monthly billing overview
